# Requirements

- Bash (included in Git for Windows)
- Python 3
- Visual Studio Code or equivalent

# Development

with system (zach)

```sh
edit  cafe.subhelion.com
```

with system

```sh
clone subhelion/cafe.subhelion.com
edit  subhelion/cafe.subhelion.com
```

without system

```sh
git clone git@github.com:subhelion/cafe.subhelion.com.git
cd cafe.subhelion.com
code .
```
